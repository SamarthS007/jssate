## Automatic Deployment
Your repository is configured for automatic deployment to Vercel. Every time you push to the main branch, your website will be automatically deployed.

## Manual Deployment (Alternative)
If you prefer manual deployment:
1. Go to [vercel.com](https://vercel.com) and sign up/login
2. Click "New Project"
3. Import this GitHub repository
4. Click "Deploy"

## Files Structure
- `index.html` - Your main website file
- `style.css` - Your website styling (if created)
- `vercel.json` - Vercel configuration
- `.github/workflows/vercel-deploy.yml` - Auto-deployment workflow

## Your Website
After deployment, your website will be live at:
- `https://your-repo-name.vercel.app`
